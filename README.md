# Godot procedural generation app

for Godot v4.2.2

```
+ noise generation
+ generating and rendering single mesh
+ rendering terrain out of meshes
+ zooming in/out
+ moving camera
+ create/destroy terrain meshes when camera moves
+ rotating camera
+ create terrain meshes in separated thread
```
